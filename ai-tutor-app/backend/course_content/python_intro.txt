INTRODUCTION TO PYTHON PROGRAMMING LANGUAGE

What is Python?

Python is a high-level, interpreted, general-purpose, and open-source programming language designed to make programming simple, readable, and efficient.

It allows developers to write clean and logical code for small as well as large-scale projects.

Python emphasizes simplicity, readability, and productivity.

Definition:
Python is an easy-to-learn, powerful, and versatile programming language used for building software, websites, artificial intelligence systems, data analysis tools, automation scripts, and many other applications.

History of Python

Python was created by Guido van Rossum.
It was first released in 1991.
It was developed at Centrum Wiskunde and Informatica (CWI), Netherlands.
The name Python was inspired by the comedy show "Monty Python’s Flying Circus".

Major Python Versions:
Python 1.0 - 1991
Python 2.0 - 2000
Python 3.0 - 2008
Latest versions: Python 3.x (actively updated)

Note: Python 2 is discontinued. Python 3 is the standard version used today.

Python Language Characteristics

Interpreted Language:
Python executes code line by line, which makes debugging easy.

Dynamically Typed:
Data types do not need to be declared explicitly.

Object-Oriented:
Supports classes, objects, inheritance, encapsulation, and polymorphism.

Python Program Structure

Python Modules and Packages

Module:
A file containing Python code is called a module.

Package:
A collection of modules is called a package.

Example:

print(math.sqrt(25))

Python File Handling

Python allows reading and writing files.

Example:

print(file.read())
file.close()

Python Exception Handling

Exception handling is used to manage runtime errors.

Example:

Easy to learn.

High productivity.

Large library support.

Limitations of Python

Higher memory consumption.

Not ideal for mobile and low-level programming.
