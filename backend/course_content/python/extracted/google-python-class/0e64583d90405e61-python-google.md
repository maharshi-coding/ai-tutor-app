مرحبًا بك في Python Class من Google -- هذا الفصل الدراسي مجاني للأشخاص الذين لديهم خبرة قليلة في البرمجة يرغبون في تعلم Python. يتضمن الصف موادًا مكتوبة ومقاطع فيديو للمحاضرات والكثير من التمارين المتعلقة بالترميز للتمرُّن على ترميز Python. يتم استخدام هذه المواد في Google لتقديم Python إلى الأشخاص الذين لديهم خبرة قليلة في البرمجة. تعمل التمارين الأولى على مفاهيم Python الأساسية مثل السلاسل والقوائم، مما يؤدي إلى التدريبات اللاحقة، وهي عبارة عن برامج كاملة تتناول الملفات النصية والعمليات واتصالات http. تم إعداد هذا الصف للأشخاص الذين لديهم خبرة قليلة في البرمجة ببعض اللغات، وهو ما يكفي لمعرفة ما يعنيه "المتغيّر" أو "if if". وبخلاف ذلك، لا يلزم أن تكون مبرمجًا خبيرًا لاستخدام هذه المادة.

للبدء، يتم ربط أقسام Python على يمين الصفحة -- Set Python (إعداد Python) لتثبيت Python على جهازك، أو Python Introduction (مقدّمة Python) للحصول على مقدمة عن اللغة، ثم Python Strings (سلاسل Python) تبدأ مادة الترميز وتؤدي إلى التمرين الأوّل. تتضمن نهاية كل قسم مكتوب رابطًا لتمرين الشفرة في مواد هذا القسم. توازي مقاطع الفيديو التي تقدم المحاضرات المواد المكتوبة، وتقدم لغة Python، ثم السلاسل ثم التمارين الأولى وهكذا. في Google، تشكّل جميع هذه المواد صفًا مكثّفًا مكوّنًا من يومين، وبالتالي يتم تنظيم مقاطع الفيديو كقسمين لليوم الأول والثاني.

تم إنشاء هذه المادة بواسطة {0} Nick Parlante {/ 1} وهو يعمل في مجموعة engEDU في Google. شكرًا خاصًا على المساعدة من زملائي في Google جون كوكس وستيف غلاسمان وبيوتر كامينسكي وأنطوان بيكارد. وأخيرًا، أشكر Google والمديرة ماغي جونسون على كرم التنوير لنشر هذه المواد على الإنترنت مجانًا بموجب ترخيص Creative Commons Attribution 2.5 -- فشارك واستمتع!
